## 📌 Proyectos destacados

### ChambaApp — Plataforma de servicios
Aplicación web desarrollada con React, NestJS, Prisma, PostgreSQL y MongoDB.  
Incluye autenticación, roles de usuario, gestión de servicios, solicitudes y chat en tiempo real mediante WebSockets.  
Desplegada en AWS EC2 con Docker, Nginx y dominio personalizado.

### SmartMenu — Sistema para restaurante
Sistema multiplataforma desarrollado con .NET MAUI, Node.js y MySQL.  
Incluye gestión de mesas, pedidos, cocina, inventario, proveedores, roles de usuario y reportes.

### Pet Radar — Backend geoespacial
API backend desarrollada con NestJS, PostGIS, Redis, Docker y Azure Application Insights.  
Incluye endpoints con caché, monitoreo y despliegue mediante contenedores.
